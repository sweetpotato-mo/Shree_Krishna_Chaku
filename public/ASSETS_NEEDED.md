# Required Assets

Please add the following assets to this `public/` directory:

## Video
- `chaku-pulling.mp4` - Slow-motion video loop of Chaku being pulled on a wooden peg (for Hero section)

## Images

### Factory
- `factory-tokha.jpg` - Atmospheric, sepia-toned image of the factory in Tokha-3

### Products
- `sada-chaku.jpg` - Sada Chaku product image
- `masala-chaku.jpg` - Masala Chaku product image  
- `khuwa-chaku.jpg` - Khuwa Chaku product image
- `sutkeri-chaku.jpg` - Sutkeri Chaku product image
- `placeholder-chaku.jpg` - Fallback product image (optional, used if specific product images aren't available)

## Image Specifications

- **Hero Video**: 1920x1080 minimum, MP4 format, optimized for web
- **Factory Image**: 1200x800px recommended, sepia-toned aesthetic
- **Product Images**: 800x600px recommended, high quality, consistent lighting

## Notes

- All images should be optimized for web (compressed but high quality)
- Consider using WebP format for better performance
- Ensure images reflect the heritage and artisanal nature of the brand
